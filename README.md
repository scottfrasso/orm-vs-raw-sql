# ORM vs Raw SQL

You need to open this up with [VS Code within a dev container](https://code.visualstudio.com/docs/devcontainers/tutorial) to be able to run this code.

## Setup the DB

Generate the prisma client classes

`yarn generate`

Migrate the DB

`yarn migrate`

Seed the DB

`yarn seed`

## With Prisma

`yarn with-prisma`

## Without prisma

`yarn raw`
